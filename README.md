# Nutrigest - Food Tracker for iOS

Nutrigest makes food tracking and calorie counting easy.

## Features:

- set goals: lose weight, gain weight or maintain
- track calories, water intake, and meal history
- access meal plans and healthy recipes
- log activity and exercise
